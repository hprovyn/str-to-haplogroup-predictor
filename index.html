Haplogroup Predictor


Developed by Hunter Provyn with input and financial support from Thomas Krahn of YSEQ.
The haplogroup prediction is backed by a Random Forest model implemented in python using sklearn.

https://github.com/hprovyn/str-to-haplogroup-predictor

Enter STRs in one of two formats, then press ENTER:

$STR1=$ALLELE1,$STR2=$ALLELE2, ... OR $STR1 $ALLELE1 $STR2 $ALLELE2 ...
